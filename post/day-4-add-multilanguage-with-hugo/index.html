Day 4 - Add Multilanguage with Hugo

Jan 14, 2017 #100daysofcode #hugo

Actually, there is not many things to talk about Hugo multilanguages support cos everything was described in details in multilangual documentation

Only one thing I want to add here is if you want to detect user region automatically for displaying the right language then you can use ip info service.

	$.getJSON('http://ipinfo.io', function(data){
  	console.log(data)
	})

then you will get the response for the ip info which contains the location information:

	{
	  "ip": "111.223.103.130",
	  "hostname": "No Hostname",
	  "city": "Nepal Park",
	  "region": "SINGAPORE",
	  "country": "SG",
	  "loc": "1.3017,103.7881",
	  "org": "AS4657 StarHub Internet Exchange"
	}
